Un gestionnaire de marque-pages web développé avec **Symfony 7** et **Tailwind CSS**. Ce projet permet de centraliser des ressources web (URLs) avec des commentaires associés, tout en automatisant la gestion temporelle des entrées.

## 🚀 Fonctionnalités
- 📋 **Affichage dynamique** : Liste des liens récupérés depuis une base de données MySQL.
- 🎨 **Interface moderne** : Design épuré utilisant Tailwind CSS (via CDN).
- ⚙️ **Architecture Pro** : Utilisation du composant Doctrine ORM pour la persistance des données.
